##### Tahapan mengolah Ayam Sosis Wortel Kuah Kecap:

1. Siapkan semua bahan. Cuci terlebih dahulu ayam fillet hingga bersih. Kemudian iris sosis dan wortel.
1. Panaskan teflon/wajan dengan api sedang. Kemudian kecilkan api dan panggang ayam hingga natang.
1. Setelah ayam matang, masukkan wortel dan sosis. Lalu rendam dengan air secukupnya. Naikkan api menjadi api sedang.
1. Masukkan cawe rawit iris, kecap manis, saos teriyaki, bawang putih bubuk, aduk hingga merata, kemudian naikkan api sedikit lagi supaya kuah cepat mendidih.
1. Koreksi rasa bila diperlukan kemudian matikan api jika sudah dirasa cukup. Tuang ke dalam wadah saji untuk siap disantap bersama nasi.
